# 아이디어 백로그 (자유 저장소)

떠오르는 대로 적는 곳. 정리는 나중에. (실제 작업 추적은 GitHub Issues 권장)

## 시각/연출
- 광고 "우주쓰레기" 푯말 (10 DT 거리, 0.5 DT 크기, 클릭→광고) — 비영리면 제외
- 초신성급 화제: 멀리서도 보이는 플레어
- 거대 별 뒤 + 작은 별 앞 통과 (깊이감)
- 별 핀(북마크): 지지 않게 고정

## 데이터/알고리즘
- 다리(bridge) 논문: Leiden 군집화 + 매개중심성/참여계수
- 위치=유사도 배치: 임베딩(SPECTER) → UMAP/t-SNE로 3D 좌표
- 인용수 편향 보정(분야·연도 정규화), "급부상(rising)" 지표
- 언어 토글(ko/en/ja…)

## 모드/경험
- 우주·밤하늘·스톤헨지·학술을 한 앱에서 전환
- 학술 우주(OpenAlex) 프로토타입
- 타임라인 재생(분야가 자라는 영상)

## 사업/운영
- 호스팅: GitHub Pages(정적, 무료, 매일 자동 갱신)
- 상표(서비스명/로고) 가볍게 챙기기 / 특허는 검증 후
- 해자 = IP보다 실행속도·커뮤니티·경험 설계

## 거리 단위 메모
- 1 DT = 1초 직진 거리(기본). 행성간 평균거리 등 DT로 표기.
