# [Issue 초안] 별 표현/구분 방법 정리 (시각 인코딩)

라벨: `design`, `enhancement`, `discussion`

> 별 하나가 여러 데이터 차원(중요도·모멘텀·분야·최신성·관계·특수상태)을 가질 때,
> 그걸 "어떤 시각 채널"로 보여줄지 정리한다. 실제 천문학의 별 분류 + 디자인적 지표를 함께 검토.
> 원칙: **한 뷰에서 동시에 쓰는 시각 채널은 3~4개로 제한**(과하면 안 읽힘).

## A. 실제 별 현상에서 가져오기 (과학적 근거 = 직관적)

| 천문학 개념 | 실제 의미 | 우리 데이터 매핑 | 우선순위 |
|---|---|---|---|
| 광도(절대등급) | 본질적 밝기 | 중요도(인용/조회/점수) → **글로우 밝기** | **P1** (사용 중) |
| 반지름(왜성~초거성) | 별 크기 | 중요도/질량 → **별 크기** | **P1** (사용 중) |
| 성운·헤일로 | 주변 발광 | 글로우 표현 | **P1** (사용 중) |
| 고유운동/시선속도 | 별의 이동 | 이동 속도 (핫=느림) | **P1** (사용 중) |
| 분광형 색(OBAFGKM, 청→적) | 표면 온도 | **최신성**(신=청/뜨거움, 노=적/식음) 또는 분야 | **P2** |
| 변광성(세페이드 맥동) | 주기적 밝기 변화 | **모멘텀/활성도 → 맥동(깜빡임)** | **P2** |
| 신성·초신성 | 폭발적 증광 | 급부상/바이럴 → **플레어** | **P2** |
| 회절 스파이크(허블 4·JWST 6) | 밝은 별일수록 뚜렷(망원경 특성) | 상위 별 **강조·아이코닉**(스파이크 개수/길이=등급) | **P2** |
| 점멸(대기 산란) | 반짝임 | 실시간/활성 표시 | **P2** |
| HR도(광도×온도) | 별 분류 체계 | **중요도×최신성 사분면 = 별 종류** 분류 틀 | **P3** |
| 자전에 의한 편평(oblate) | 빠른 자전체는 타원형 | 특정 분류 → **비원형(타원) 모양** | **P3** |
| 쌍성·다중성 | 중력 결합 | 강하게 묶인 쌍(연관/공저) | **P3** |
| 항성 진화(색 변화) | 일생 단계 | 신생→노화 **색 변화 애니메이션** | **P4** |

### HR도 응용 아이디어 (별 "종류" 분류)
가로=최신성, 세로=중요도(인용)로 두면 4분면이 곧 별 종류가 됨:
- 청색 주계열(신생·상승) = 최근+뜨는 / 초거성(오래+거대) = 고전 명작
- 백색왜성(오래+희미) = 잊힌 옛글 / 신생 희미 = 갓 나온 무명

## B. 디자인 전용 지표 (물리 아니지만 유용)
- **회절 스파이크 4/6/8점**: 다윗의별(6각)·십자(4각). 개수·길이로 등급 표현, 멀리서도 아이코닉.
- **고리/헤일로(토성형)**: 특수 상태 = 북마크 / 내 글(논문) / 추천·featured.
- **꼬리/혜성 trail**: 인용·조회가 늘어나는 궤적(움직임 잔상).
- **모양 다양화**: 원(기본) / 타원(분류) / 스파이크 / 다윗의별·십자.
- **불투명도**: 불확실성 또는 오래됨(노화).

## C. 우선순위 정리
- **P1 (필수·이미 사용):** 크기, 글로우 밝기, 색(분류), 이동속도
- **P2 (시간 스토리 강화):** 맥동=모멘텀, 회절 스파이크=상위 강조, 색온도=최신성, 초신성 플레어, 점멸
- **P3 (선택적):** 비원형 모양=분류, 고리=특수상태, 쌍성=강결합 쌍
- **P4 (실험):** HR 분류 사분면, 색변화 애니, 혜성 꼬리
